# Asterra Assessment Frontend

## Technologies used for this project

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS


## Setup

1. cd frontend
2. npm install
3. npm run dev

Open the site at the address Vite prints (usually http://localhost:8080).

## Features

- Add new users
- Add hobbies for users (select user from dropdown)
- Index table showing users and their hobbies
- Delete button per row (deletes hobby if present; deletes user if hobby is empty)
